
# 🔐 Azure SC-300 Labs — Identity and Access Administrator  

**Certification:** [Microsoft Certified: Identity and Access Administrator (SC-300)](https://learn.microsoft.com/en-us/certifications/identity-and-access-administrator)  
**Focus Areas:** Zero Trust · Conditional Access · PIM · SSO · Identity Governance · Automation · Hybrid Identity  

---

## 🧭 Overview
This folder contains all hands-on labs developed for the **SC-300: Microsoft Identity and Access Administrator** certification path.  
Each lab demonstrates how to design, implement, and manage secure identity solutions using **Microsoft Entra ID (Azure AD)**.

The labs are organized sequentially, following real-world security implementation flows — from enforcing Zero Trust to automating identity lifecycle and privileged access.

## 🧠 Learning Objectives
By completing these labs, you will:
- Implement **Zero Trust** identity and access strategies.  
- Configure and automate **Privileged Identity Management (PIM)**.  
- Deploy **Conditional Access** and passwordless authentication.  
- Integrate **SSO** with SAML and OIDC apps.  
- Govern identities through **Access Reviews** and **Lifecycle Workflows**.
